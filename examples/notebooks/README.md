# Mother Tutorial Notebooks

Welcome to the comprehensive tutorial collection for the Mother machine learning framework! 
These notebooks provide hands-on guidance for molecular property prediction, cheminformatics, and advanced ML techniques.

## 🎯 Learning Path

### For Beginners: Start Here
If you're new to the Mother framework, follow this recommended learning sequence:

1. **📚 Framework Basics** → `01_basics/01_mother_framework_overview.ipynb`
2. **⚙️ Configuration** → `01_basics/02_configuration_settings.ipynb`
3. **🔬 Simple Regression** → `02_regression/01_basic_regression.ipynb`
4. **🎯 Binary Classification** → `03_classification/02_binary_classification_uncertainty.ipynb`

### For Intermediate Users
Once comfortable with basics, explore specialized techniques:

5. **🏗️ Custom Preprocessing** → `01_basics/03_custom_preprocessing.ipynb`
6. **🌲 Random Forest Models** → `02_regression/02_random_forest_regression.ipynb`
7. **📊 Feature Selection** → `04_feature_engineering/01_feature_selection.ipynb`
8. **🎲 Uncertainty Quantification** → `02_regression/03_regression_with_uncertainty.ipynb`

### For Advanced Users
Dive into sophisticated methods and specialized algorithms:

9. **🔧 Hyperparameter Optimization** → `05_advanced/01_custom_hyperparameter_optimization.ipynb`
10. **🧬 TabPFN Integration** → `05_advanced/02_tabpfn_integration.ipynb`
11. **📈 Ordinal Classification** → `03_classification/03_ordinal_classification.ipynb`
12. **🧪 Multi-task Learning** → `02_regression/04_multitask_random_forest.ipynb`
